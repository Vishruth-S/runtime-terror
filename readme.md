## How to run locally
1. Clone this repo
```
git clone https://github.com/Vishruth-S/runtime-terror.git
```
2. Navigate to the project folder
```
cd runtime-terror/devcraft-website
```
3. Install dependencies
```
npm install
```
4. Start the server 
```
npm start
```
